# Android-task-reminder
App for creating and managing task reminders with support for attaching images or screenshots.

## Overview
TaskReminder helps users manage daily tasks with reminders, subtasks, voice input, image attachments, location-based alerts, and a built-in Pomodoro timer — all wrapped in a clean UI with dark mode support.

## Features

| Feature | Details |
|---|---|
| Task Management | Add, edit, delete tasks with title, description, due date/time |
| Priority System | High / Medium / Low with color indicators |
| Categories & Tags | Custom color-tagged categories for task grouping |
| Subtasks | Add checklist items within any task |
| Image Attachment | Attach photos via camera or gallery |
| Voice Input | Dictate task title or description via microphone |
| Time Reminder | Exact alarm notification at task due time |
| Location Reminder | Geofence-based notification when near a set location |
| Repeat Tasks | Daily / Weekly / Monthly auto-rescheduling |
| Streak Tracking | Habit streak counter for recurring tasks |
| Pomodoro Timer | Built-in focus timer linked to tasks |
| Analytics Dashboard | Completion rate, priority split, weekly progress charts |
| Smart Search & Filter | Filter by priority, category, date range, status |
| Export | Export task list to PDF or CSV |
| Dark Mode | Toggle dark/light theme, persists across sessions |
